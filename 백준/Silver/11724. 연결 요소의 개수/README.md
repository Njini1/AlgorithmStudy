# [Silver II] 연결 요소의 개수 - 11724 

[문제 링크](https://www.acmicpc.net/problem/11724) 

### 성능 요약

메모리: 142092 KB, 시간: 572 ms

### 시간 및 난이도
- 하
- 10분

### 풀이 리뷰
- 손풀기로 간단히 DFS 풀어봄

### 풀이 키워드
- DFS

### 풀이 방법
1. 인접 리스트로 그래프를 만듦
2. 1번 정점부터 N번까지 순회하면서 방문하지 않은 정점을 시작점으로 DFS
3. 연결된 곳 중 visited[i] == false 인 곳을 순회하면서 visited 값을 true로 변경
4. 모든 연결된 요소를 찾았으면 count++
5. 다시 또 방문하지 않은 정점을 시작점으로 DFS

### 시간 복잡도
- O(N + M)
    - 정점 방문 N + 간선 방문 2M

### 분류

그래프 이론, 그래프 탐색, 너비 우선 탐색, 깊이 우선 탐색

### 문제 설명

<p>방향 없는 그래프가 주어졌을 때, 연결 요소 (Connected Component)의 개수를 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 정점의 개수 N과 간선의 개수 M이 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ M ≤ N×(N-1)/2) 둘째 줄부터 M개의 줄에 간선의 양 끝점 u와 v가 주어진다. (1 ≤ u, v ≤ N, u ≠ v) 같은 간선은 한 번만 주어진다.</p>

### 출력 

 <p>첫째 줄에 연결 요소의 개수를 출력한다.</p>
